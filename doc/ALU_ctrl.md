# ALU 控制真值表（本工程）

下表给出主控输出 ALUOp 与 RISC-V R 型指令 funct7/funct3 到 ALU 控制码 alu_control[3:0] 的映射（与 `rtl/ALU.v` 实现一致）。

- ALUOp[1:0] 由主控制器产生：
	- 00：加法类（如 LOAD/STORE 地址计算）
	- 01：分支等需要减法比较（如 BEQ 走 SUB+Zero）
	- 10：R 型，依据 funct7/funct3 精细选择
- Operation 列为 ALU 控制码（4 位）：
	- 0000: ADD
	- 0001: SUB
	- 0011: AND
	- 0100: OR
	- 0101: XOR
	- 0110: SLT（有符号）
	- 0111: EQ（相等置 1；可选，用于直接产生等于结果）

| ALUOp1 | ALUOp0 | funct7[5] | funct3[2] | funct3[1] | funct3[0] | Operation | 含义 |
|:------:|:------:|:---------:|:---------:|:---------:|:---------:|:--------:|:-----|
|   0    |   0    |     X     |     X     |     X     |     X     |  0000    | 加法（地址计算等） |
|   X    |   1    |     X     |     X     |     X     |     X     |  0001    | 减法（分支比较 SUB+Zero） |
|   1    |   X    |     0     |     0     |     0     |     0     |  0000    | R 型：ADD（funct7=0000000, funct3=000） |
|   1    |   X    |     1     |     0     |     0     |     0     |  0001    | R 型：SUB（funct7=0100000, funct3=000） |
|   1    |   X    |     X     |     1     |     1     |     1     |  0011    | R 型：AND（funct3=111） |
|   1    |   X    |     X     |     1     |     1     |     0     |  0100    | R 型：OR（funct3=110） |
|   1    |   X    |     X     |     1     |     0     |     0     |  0101    | R 型：XOR（funct3=100） |
|   1    |   X    |     X     |     0     |     1     |     0     |  0110    | R 型：SLT（有符号，funct3=010） |

说明：
- X 表示“不关心”。
- 若采用“通过 SUB 并查看 Zero”实现 BEQ/ BNE，则无需使用 0111(EQ) 控制码；如希望直接得到等于结果，可为相等比较另行分配 0111。
